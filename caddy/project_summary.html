專案概述 & 技術特點

本專案旨在本地端實作一個基於 Caddy 與 Cloudflare Tunnel 的高性能轉址閘道。我們為其打造了動態規則編輯、安全認證及狀態掌控的 Python 管理平台。

🔗 自動化 Cloudflare DNS/Ingress

全自動註冊 CNAME DNS 紀錄，並透過 Cloudflare API 同步 Ingress 路由，無須登入後台手動操作。

🔄 Caddy 規則熱重載 (Hot-Reload)

當您在後台新增規則時，Caddyfile 會自動重新編譯並熱載入，現有連線不中斷，即時生效。

📁 本地實體空間目錄對接

支援將特定網址路徑直接對應到 Mac 本地的絕對路徑或家目錄 (~)，作為靜態網頁伺服器。

🔐 密碼保護與連線日誌

公網訪問強制驗證（falo / force），底層自動解析 JSON 存取日誌，提供排序、過濾與 CSV 匯出。

運作管理指令集

A. 手動啟動所有服務（背景常駐 + Caddy + Tunnel）：

cd /Users/force/Google_Antigravity/Cloudflare_falo/caddy && ./start.sh

B. 手動優雅關閉所有服務：

cd /Users/force/Google_Antigravity/Cloudflare_falo/caddy && ./stop.sh

C. 安裝為 macOS 系統開機登入自啟動 (launchd)：

cp ./launchd/com.falo.caddy-admin.plist ~/Library/LaunchAgents/
